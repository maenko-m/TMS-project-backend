defects

Description

Retrieves a paginated, sortable list of all defects in the system with optional filtering.

Returns:

A queryable collection of DefectOutputDto representing the defects.

Response

Returns a DefectsConnection

Arguments
Name Description
filter - DefectFilterInput Optional filtering criteria for narrowing down results.
first - Int Returns the first n elements from the list.
after - String Returns the elements in the list that come after the specified cursor.
last - Int Returns the last n elements from the list.
before - String Returns the elements in the list that come before the specified cursor.
order - [DefectOutputDtoSortInput!]

Example

Query
query defects(
  $filter: DefectFilterInput,
  $first: Int,
  $after: String,
  $last: Int,
  $before: String,
  $order: [DefectOutputDtoSortInput!]
) {
  defects(
    filter: $filter,
    first: $first,
    after: $after,
    last: $last,
    before: $before,
    order: $order
  ) {
    pageInfo {
      ...PageInfoFragment
    }
    edges {
      ...DefectsEdgeFragment
    }
    nodes {
      ...DefectFragment
    }
  }
}
